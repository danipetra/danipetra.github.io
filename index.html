October 27, 2022

VR Graph Viewer

Unity

For my thesis work, I combined my passion for Unity and game development with my experience in data analysis and visualization: I created a VR visualization app for showing network anomalies using force-directed graphs, with high emphasis on interaction and anomaly detection. To recover network traffic data records before the rendering, I developed a Python Restful API, using pandas and flask. The main feautes this work offers are:
- Player free movement and camera control.
- High interaction with nodes.
- Anomaly detection, using the MIDAS library.
- Community detection, using the Comuna library.
- Graph controls (as rotation or resizing).
- High performance on small-medium sized graphs.
